Lesson 1: HTML and the web

HTML is an important part of how the internet works

HTML is Hyper Text Markup Language. It is the basis of most of the documents on the web. HTML documents contain text content, for how the text should be displayed on the webpage

When you use your computer to go to a website, your computer uses something called Hyptertext Transfer Protocol or HTTP to send requests to servers.

HTML markup basics: tags,elements,and attributes

Tags: These are basically instructions inserted into the HTML document that give directions on how it should be formatted.

Elements: Individual components of HMTL documents. For example, you can set height and width.

Attributes: These are properties or values of an HTML element that modify it. For example, "href" is the attribute that tells the browswer that the text content of the element should be http//alivelinktoawebsite
